# Weekly CEO Briefing

Generate a comprehensive Monday morning briefing summarizing business performance.

## Instructions

1. **Review Business_Goals.md** (if exists) for targets
2. **Check /Done/** for completed tasks this week
3. **Check /Accounting/** for financial transactions
4. **Check /Logs/** for all activities
5. **Identify bottlenecks** and issues
6. **Generate briefing** in `/Briefings/`

## Data Sources

### Check These Files/Folders

1. **Business_Goals.md** - Revenue targets, KPIs
2. **/Done/** - All completed tasks
3. **/Needs_Action/** - Still pending
4. **/Accounting/*.md** - Financial records
5. **/Logs/{{dates}}.md** - Activity logs
6. **Dashboard.md** - Current state

## After Generating

1. Save to `/Briefings/YYYY-MM-DD_Monday_Briefing.md`
2. Update Dashboard.md with briefing link
3. Log briefing creation
4. Optionally suggest actions based on findings

## If Data Is Missing

If some data doesn't exist yet:
- Note it in the briefing: "No accounting records found"
- Suggest setting up the missing component
- Still generate the briefing with available data
